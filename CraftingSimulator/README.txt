=====================================================================
 NickWIll86's Just For Fun Crafting Simulator
=====================================================================

HOW TO RUN (no install needed):
  1. Unzip this folder anywhere on your computer.
  2. Double-click index.html — it opens in your web browser.
  3. That's it. Everything runs 100% offline; no internet,
     no server connection, nothing to install.

WHAT'S INSIDE:
  - Full resource list (14,536 named resources with real stats)
  - All draft schematics for 9 crafting professions
  - All 2,197 lootable items with level-scaled stat rolls
  - Satellite maps for all 10 resource planets
  - Exact server crafting math: assembly, experimentation,
    hand sampling, and loot rolls

Open the "Help" tab in the app for a full guide.
