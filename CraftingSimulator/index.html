Help Guide — Welcome, crafter!

This simulator is hooked directly into your real server files. Every resource, schematic, loot item, stat range and formula was extracted from your C:\Companion SWGEmu (Core3) server and its TRE archives — including the satellite maps you see on the Map tab. Nothing is faked: the assembly rolls, experimentation math, loot rolls and hand-sampling yields run the same code your server runs.

1 · Resource Map (surveying & hand sampling)

Pick a planet, then pick a resource from the list (it shows everything that can spawn there — planet-locked resources plus galaxy-wide ones). The colored overlay is the resource's concentration map, generated with the exact Simplex-noise algorithm from SpawnDensityMap.h: blue = trace, green = good, yellow/red = rich.

Hover the map to read the concentration % at any coordinate — the same number a survey tool would show in game. Click to plant your sampling flag, set your Surveying skill (Master Artisan = 100+), and press Sample to hand-sample exactly like standing in the dirt spamming /sample. Successful pulls are added to your inventory.

One honest note: each live spawn's noise seed is rolled when the resource shifts in and is stored inside the server's object database, so this tool can't know the current live shape — but the algorithm, density ranges and behavior are identical. Use Re-roll shift to explore possible spawn shapes, or type a seed to share a map with friends.

2 · Resources tab

The full list of named resources seeded by your server — over fourteen thousand. Search by name, class or type; filter by planet. Stats are colored against the class caps from your server's resource tree (gold = capped, green = near cap). Selecting a resource on the Map tab also shows "Best used for" — the schematics whose experimentation lines benefit most from that resource's stats.

3 · Profession tabs & crafting

Each profession tab lists every draft schematic granted by that profession's skill tree (straight from skills.iff and schematic_group.iff). Click a schematic to see its ingredient slots, resource weighting (e.g. CD 66% + OQ 33%) and experimental ranges, then hit Start Crafting Session.

The crafting window works like the in-game tool: fill each socket from your inventory (resources need enough units; component slots take looted or crafted parts), then Assemble. Your assembly roll uses the server formula with your buffed skills. Then experimentation: allocate points into property lines (boxes = points), press Experiment, and watch each line's blue bar climb toward its red cap marker — the cap is set by your ingredients' weighted stats, exactly as in game. Finish with Create Prototype.

4 · Inventory & looted parts

Your inventory looks and works like the in-game bag: an icon grid with quantity badges, hover tooltips with full stats. Add resource stacks directly, or open Get looted crafting part to browse every loot item on your server. Choose the loot level (drag up to 350 — 300 is the classic top-tier drop level) and a rarity tier (Experimental / Enhanced / Exceptional / Legendary); the item's stats are rolled with your server's LootValues level-scaling math, so a level-300 Exceptional part rolls like the real thing.

5 · Buffs (the "maxed crafter")

By default the simulator crafts as a fully maxed crafter: master profession skill mods (summed from every skill box in skills.iff), +25 assembly & +25 experimentation from clothing/armor attachments (the server's wearable skill-mod cap), a +12 Bespin Port (experimentation food buff), a +10 Pyollian Cake (assembly food buff), and a max-crafted tool at +15 effectiveness. Toggle any of these on the Buffs tab, or type exact skill values to test other loadouts. Entertainer Inspiration only boosts crafting XP on Core3 servers, not crafting rolls — the guide on that tab explains why.

Tips

• Experimentation success is heavily weighted by luck rolls — even a maxed crafter gets ~5% amazing successes. Use several small experiments (1-2 points) on important lines to reduce risk, exactly like in game.
• The red cap marker on each experimentation bar is weighted resource stats ÷ 1000 — better resources literally raise your ceiling. Chase caps on the Resources tab.
• Component slots marked (optional) can be skipped, but good looted parts in optional slots add stats the same way applyComponentStats does on the server.
• Re-roll a loot part a few times — level scaling uses a distributed random curve, not a flat roll.
